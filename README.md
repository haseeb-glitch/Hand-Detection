Hand Tracking using MediaPipe

This project is a real-time Hand Tracking application built with Python, OpenCV, and MediaPipe. It detects and tracks hand landmarks through a webcam, draws the hand skeleton on the video feed, and displays the current FPS (Frames Per Second).

Features
Real-time hand detection and tracking
Draws hand landmarks and connections
Displays live FPS
Built using Python, OpenCV, and MediaPipe
Technologies Used
Python
OpenCV
MediaPipe
Run
pip install -r requirements.txt
python main.py
